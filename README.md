# DataStructuresAndAlgorithms
Learn DataStructures &amp; Algorithms using Golang

## sorter

Go to sorter folder and run ```./Build``` to build sorter programe in ```bin``` folder.

Run```./sorter```

- ```-a``` assign AndAlgorithm
- ```-i``` assign input file
- ```-o``` assign output file

Sample: 

```bash
cd bin

./sorter -i unsorted.dat -o sorted.dat -a shellsort
./sorter -a bubblesort
./sorter -a insertionsort
./sorter -a selectionsort
./sorter -a shellsort
```
